常用hexo命令

常见命令

hexo new “文件名” #新建文件名
hexo new page “页面名” #新建页面
hexo generate #生成静态页面至public目录下
hexo server #开启预览访问端口
hexo deploy #部署到GitHub
hexo help #查看帮助文档
hexo version #查看Hexo的版本

缩写

hexo n == hexo new
hexo g == gexo generate
hexo s == hexo server
hexo d == hexo deploy

组合命令

hexo s -g #生成并本地预览
hexo d -g #生成并上传

写博客

定位到hexo根目录，执行命令

hexo new “文件名”
hexo会帮我们在_posts下生成相关的md文件
我们只需要打开文件写博客就行了

一般完整格式:

title: 文章页面上显示的名称
date: 2018-11-24 20:21:32
tags: 文章标签
categories: 默认分类
description: 附加一段文章摘要
